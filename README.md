# Food Order Web App

This is a full-stack web application for ordering food online. The project includes both frontend and backend components.

## Features

- Browse food items
- Add items to cart
- Place orders
- User authentication
- Order history

## Technologies Used

### Frontend
- React
- CSS

### Backend
- Node.js

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/food-order-web-app.git
    ```

2. Navigate to the project directory:
    ```bash
    cd food-order-web-app
    ```

3. Install frontend dependencies:
    ```bash
    cd frontend
    npm install
    ```

4. Install backend dependencies:
    ```bash
    cd ../backend
    npm install
    ```

## Usage

1. Start the backend server:
    ```bash
    cd backend
    npm start
    ```

2. Start the frontend development server:
    ```bash
    cd ../frontend
    npm start
    ```

3. Open your browser and navigate to `http://localhost:4000`.
